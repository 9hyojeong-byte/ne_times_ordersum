# NE times 외부주문내역 merger 사용 안내서 (readme.txt)

본 서비스는 나이스북, 북콤파스, 북매거진 등 다양한 외부 주문처의 엑셀 데이터를 하나의 통합된 형식으로 병합하고 변환해주는 도구입니다.

---

### 1. 주요 기능
- **데이터 통합**: 서로 다른 양식의 엑셀 파일을 하나의 표준화된 양식으로 자동 변환합니다.
- **상품번호 자동 매핑**: 특정 상품명(엔이 타임즈, 엔이타임즈 주니어, 키즈 등)을 정해진 상품번호(2544, 2546, 2548)로 자동 변환합니다.
- **날짜 정규화**: 각기 다른 날짜 형식을 표준 형식(YYYY-MM-DD)으로 통일합니다.
- **스마트 병합**: 중복된 데이터를 관리하고, 한꺼번에 엑셀로 다운로드할 수 있습니다.

---

### 2. 사용 방법
1. 엑셀에서 필요한 데이터 영역을 복사(Ctrl+C)합니다.
2. 서비스 상단의 **"엑셀 데이터를 붙여넣기 해주세요"** 입력창에 붙여넣습니다(Ctrl+V).
3. 우측의 **Apply data**을 누르면 데이터가 즉시 인식됩니다.

---

### 3. 결과 확인 및 다운로드
- 하단 리스트에서 인식된 데이터를 확인할 수 있습니다.
- 잘못 들어온 데이터는 각 행 우측의 ** X 버튼**으로 삭제 가능합니다.
- 모든 준비가 끝나면 우측 상단의 **"엑셀로 다운받기 (.xlsx)"** 버튼을 누릅니다.

---

### 4. 파일명 규칙
다운로드되는 파일은 다음과 같은 규칙으로 자동 명명됩니다:
`주문건합본_인식된사이트명_날짜(YYMMDD).xlsx`
예) 주문건합본_나이스북_북콤파스_260430.xlsx

---

### 5. 참고 사항
- **정산구분**: 모든 데이터의 정산구분은 자동으로 "GARA2"로 지정되나, 엔이타임즈 관리자페이지에 업로드하면 "능률"로 인식 됩니다.
- **데이터 보안**: 본 서비스는 사용자의 데이터를 서버에 저장하지 않으며, 모든 처리는 브라우저 내에서 안전하게 이루어집니다.
